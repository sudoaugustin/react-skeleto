## Installation

Install the package into your project via command line.

```package-install
@skel-ui/react
```

## Importing CSS

Import the CSS file into the root of your application.

```js
import "@skel-ui/react/styles.css";
```

Now you not only have designed the skeleton, you have also finished your real UI. The above code will render a skeleton while fetching and an email address when fetching is complete. See live example here.

> Unlike other skeleton libraries, Skel UI only allow CSS to define the size of the skeleton for mobile responsiveness.

## Customization

Customize the default color and border-radius of skeleton using css variables.

```css title="global.css"
:root {
  --skel-ui-color: #f0f0f0;
  --skel-ui-radius: 0.5rem;
}
```
