## generatePlaceholder

```js
const placeholderUsers = generatePlaceholder(length, primaryKey);
```

### Parameters

| Name         | Type     | Description                                                         |
| ------------ | -------- | ------------------------------------------------------------------- |
| `length`     | `number` | The length of placeholder                                           |
| `primaryKey` | `string` | Name of the attribute that will be unique and will be used as `key` |
