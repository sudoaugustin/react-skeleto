---

## Import plugin

Add Skel UI plugin to the plugins array of your tailwind config file.

```js title="tailwind.config.js"
/** @type {import('tailwindcss').Config} */

module.exports = {
  plugins: [
    require("@skel-ui/react/tailwind"), // [!code highlight]
  ],
};
```

## Variants

### `loading`

This variant is used to style the element while `isLoading=true`. Useful to define the width and height of the skeleton.

```json doc-gen:file
{
  "file": "./codes/tailwind-css/UserProfileLoading.jsx",
  "codeblock": {
    "lang": "jsx"
  }
}
```


### `loaded`

This variant is used to style the element after `isLoading=false`. Useful to define after effects like applying borders and hover effects.

```json doc-gen:file
{
  "file": "./codes/tailwind-css/UserProfileLoaded.jsx",
  "codeblock": {
    "lang": "jsx"
  }
}
```
