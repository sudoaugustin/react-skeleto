## Root

Handles the loading state of all children `Item` using context.

```jsx
import Skel from "@skel-ui/react";

function Demo() {
  return (
    <Skel.Root isLoading={isLoading}>
      <Skel.Item>
        {/* Will render the skeleton or real UI based on `isLoading` */}
      </Skel.Item>
      <div>....</div>
    </Skel.Root>
  );
}
```

### Props

| Name        | Type    | Default | Description                          |
| ----------- | ------- | ------- | ------------------------------------ |
| `isLoading` | boolean | `true`  | Show the loading skeleton if `true`. |

## Item

Takes the loading state from nearest `Root` component and render the UI accordingly.

```jsx
import Skel from "@skel-ui/react";

function Demo() {
  return (
    <Skel.Root isLoading={isLoading}>
      <Skel.Item>...</Skel.Item>
    </Skel.Root>
  );
}
```

### Props

| Name       | Type                             | Default | Description                                  |
| ---------- | -------------------------------- | ------- | -------------------------------------------- |
| `as`       | `ElementType \| ReactElement`    | `p`     | HTML tag or React component to render        |
| `children` | `ReactNode \| (() => ReactNode)` | -       |                                              |
| `color`    | `string`                         | -       | Overrides the default skeleton color         |
| `radius`   | `string`                         | -       | Overrides the default skeleton border-radius |

### Custom tag

To render custom HTML tag, please use `as` property.

```jsx
<Skel.Item as="a" href="https://skel-ui.augustin.zip">
  ...
</Skel.Item>
```

### Custom component

When you pass a component, all of its props are sent to the `Item` component.

```jsx
import Image from "next/image";

<Skel.Item as={Image} src="..." width={400} height={500} quality={85} />;
```
